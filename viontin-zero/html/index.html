Zero Config

Single binary, no DSL, no boilerplate. Start with one file, scale to any architecture without changing the framework.

📐

6 Levels of Scale

Progressive architecture from Zero to Mesh. Each level adds guardrails automatically detected from your project structure.

🎨

CLI Toolkit

Artisan-inspired command system with styled output, interactive prompts, spinners, and progress bars — Laravel ergonomics for Rust.

🌐

Built-in HTTP

Zero-dependency HTTP server with file-based routing. No tokio, no hyper, no axum — just Rust's standard library.

🗄️

Multi-Driver ORM

Full ORM with migrations, schema builder, and relations. PostgreSQL, MySQL, and SQLite support out of the box.

🧩

Plugin System

Extend with gems — integrations, platforms, databases, and dev tools. WASM-based dynamic loading for third-party plugins.
